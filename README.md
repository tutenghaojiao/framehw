#             框架简介
### 文件目录
                1.  app         开发者写代码的地方

                    home        前台模块(controller控制器类 和  view视图两个文件目录)
                    
                2.  test        系统核心（框架）   
                    cor         核心类目录（Boot.php核心类文件）

                    static       静态资源
                    view         公共模版文件(message.php ,welcome.php 欢迎、提示性模版)
                    index.php    单一入口文件
                 
                4.system         配置项文件
                    config       配置项文件目录
                    model        处理业务的各中模型类
